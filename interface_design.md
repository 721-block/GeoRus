# Дизайн интерфейса
## Приложение для разработки маршрутов "Майской прогулки"

### 1. Общие принципы дизайна

#### 1.1 Визуальная концепция
- **Минимализм**: Чистый интерфейс без лишних элементов
- **Функциональность**: Каждый элемент имеет четкое назначение
- **Интуитивность**: Понятные и знакомые паттерны взаимодействия
- **Профессионализм**: Сдержанная цветовая гамма и типографика

#### 1.2 Цветовая палитра
- **Основные цвета**: Глубокий зеленый (#2D5A27), теплый серый (#6B7280)
- **Акцентные цвета**: Янтарный (#F59E0B) для активных элементов
- **Фоновые цвета**: Светло-серый (#F9FAFB) и белый (#FFFFFF)
- **Цвета состояний**: Зеленый для успеха, красный для ошибок, синий для информации

#### 1.3 Типографика
- **Основной шрифт**: Inter для заголовков и UI элементов
- **Шрифт для данных**: JetBrains Mono для чисел и координат
- **Иерархия**: Четкое разделение на уровни заголовков и текста

### 2. Структура интерфейса

#### 2.1 Основная компоновка
```
┌─────────────────────────────────────────────────────────────┐
│  Навигационная панель (60px)                               │
├─────────────┬───────────────────────────────────┬───────────┤
│             │                                   │           │
│  Панель     │                                   │  Панель   │
│  инструмент │        Карта                      │ свойств   │
│  (300px)    │                                   │ (350px)   │
│             │                                   │           │
└─────────────┴───────────────────────────────────┴───────────┘
```

#### 2.2 Навигационная панель
- **Логотип**: "Майская прогулка" слева
- **Главное меню**: Файл, Правка, Вид, Инструменты, Справка
- **Индикаторы состояния**: Статус сохранения, пользователь, сеть
- **Быстрые действия**: Создать, Сохранить, Экспортировать

#### 2.3 Панель инструментов
**Группа 1: Рисование**
- Карандаш (свободное рисование)
- Линия (прямые участки)
- Полилиния (свободная линия)
- Многоугольник (зоны)

**Группа 2: Редактирование**
- Выбор (перемещение точек)
- Добавить точку
- Удалить точку
- Разделить маршрут

**Группа 3: Анализ**
- Измерение расстояния
- Измерение площади
- Расчет времени
- Профиль высот

**Группа 4: Вид**
- Zoom In/Out
- Показать все
- Слои
- Спутниковый вид

#### 2.4 Панель свойств
**Вкладка 1: Общие**
- Название маршрута
- Год мероприятия
- Общая длина
- Примерное время

**Вкладка 2: Статистика**
- Разбивка по типам покрытия
- График высотного профиля
- Список узловых точек
- Варианты маршрутов

**Вкладка 3: Экспорт**
- Выбор формата (GPX/KML/KMZ)
- Параметры экспорта
- Предпросмотр
- Кнопка экспорта

### 3. Интерактивные элементы

#### 3.1 Работа с картой
- **Масштабирование**: Колесо мыши, жесты на мобильных
- **Панорамирование**: Перетаскивание мышью
- **Двойной клик**: Добавление точки маршрута
- **Правый клик**: Контекстное меню

#### 3.2 Работа с точками
- **Обычные точки**: Круги синего цвета
- **Узловые точки**: Квадраты янтарного цвета
- **Выбранная точка**: Подсветка и маркеры
- **Перетаскивание**: Изменение координат

#### 3.3 Работа с линиями
- **Предварительные маршруты**: Пунктирная линия серого цвета
- **Основные маршруты**: Сплошная линия зеленого цвета
- **Альтернативные варианты**: Разные оттенки зеленого
- **Подсветка**: При наведении увеличение толщины

### 4. Диалоги и модальные окна

#### 4.1 Создание нового маршрута
- Название маршрута (обязательное)
- Год мероприятия (выпадающий список)
- Описание (необязательное)
- Тип маршрута (предварительный/основной)
- Кнопки: Создать, Отмена

#### 4.2 Экспорт маршрута
- Выбор маршрута (если несколько)
- Формат экспорта (GPX/KML/KMZ)
- Параметры (координаты, высота, временные метки)
- Предпросмотр содержимого
- Кнопки: Экспортировать, Отмена

#### 4.3 Настройки приложения
- Единицы измерения (км/мили)
- Тип карты по умолчанию
- Автосохранение (вкл/выкл, интервал)
- Язык интерфейса
- Горячие клавиши

### 5. Состояния и индикаторы

#### 5.1 Индикаторы состояния
- **Сохранение**: Иконка дискеты, прогресс-бар
- **Загрузка**: Спиннер, процент выполнения
- **Ошибки**: Красные уведомления с описанием
- **Успех**: Зеленые уведомления

#### 5.2 Состояния элементов
- **Активный**: Янтарная рамка, подсветка
- **Неактивный**: Полупрозрачность, блокировка взаимодействия
- **Выбранный**: Толстая рамка, изменение цвета
- **Наведение**: Увеличение размера, тень

### 6. Адаптивный дизайн

#### 6.1 ПК (1920x1080 и выше)
- Полноценная трехпанельная компоновка
- Все инструменты доступны
- Поддержка горячих клавиш

#### 6.2 Планшет (768x1024 - 1366x1024)
- Скрытая панель инструментов (выезжает слева)
- Упрощенная панель свойств
- Крупные кнопки для тач-взаимодействия

#### 6.3 Мобильный (до 768px)
- Полноэкранная карта
- Всплывающие панели инструментов
- Жестовое управление
- Упрощенный функционал

### 7. Горячие клавиши

- **Ctrl+N**: Создать новый маршрут
- **Ctrl+S**: Сохранить маршрут
- **Ctrl+Z**: Отменить действие
- **Ctrl+Y**: Вернуть действие
- **Ctrl+E**: Экспортировать маршрут
- **Space**: Переключить режим выбора
- **Delete**: Удалить выбранные элементы
- **Esc**: Отменить текущее действие

### 8. Анимации и переходы

#### 8.1 Микроанимации
- Появление/исчезновение панелей (300ms ease-out)
- Изменение состояний кнопок (150ms ease-in-out)
- Перетаскивание точек (плавное движение)
- Масштабирование карты (инерция)

#### 8.2 Загрузка данных
- Скелетон-экраны при загрузке
- Прогрессивная загрузка карт
- Плавное появление элементов
- Индикация фоновых процессов

### 9. Доступность

#### 9.1 Семантическая разметка
- Правильная структура HTML
- ARIA-метки для экранных читалок
- Описания для изображений
- Логическая последовательность табуляции

#### 9.2 Контрастность
- WCAG AA соответствие (4.5:1 для текста)
- Высококонтрастный режим
- Настройка размера шрифта
- Цветовые схемы для дальтонизма
